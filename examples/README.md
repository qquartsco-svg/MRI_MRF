## full_scenario.py

저장소 루트에서 실행:

```bash
python3 examples/full_scenario.py
```

`analyze()`에 Larmor, 자기 가두기, 열 proxy, 지구–L1–달 게이트 토폴로지, 플라즈마 수송, 토로이달 형상, RF 링크, 라그랑주 L1을 한 번에 넣어 **통합 Ω·ATHENA·verdict**를 출력합니다.

개념 배경은 상위 디렉터리 [CONCEPT.md](../CONCEPT.md)를 참고하세요.
